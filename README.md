# Homebrew Tap for Recopy

[Recopy](https://recopy.pages.dev) — A modern clipboard history manager for macOS.

## Install

```bash
brew tap shiqkuangsan/recopy
brew install --cask recopy
```

## Update

```bash
brew upgrade --cask recopy
```

## Uninstall

```bash
brew uninstall --cask recopy
brew untap shiqkuangsan/recopy
```
